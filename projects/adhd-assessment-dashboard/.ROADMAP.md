## Project Summary

**Goal:** Create a friendly, interactive ADHD screening dashboard.

User submits answers → API scores responses → Dashboard displays risk assessment + disclaimer.

**Features:**
- Question-by-question UI
- Real-time feedback/scoring
- Results page with clear disclaimers
- Responsive, accessible design

**Constraints:**
- NOT a medical diagnosis tool (UI must state this clearly)
- Screening questions must be evidence-based (DSM-5, ASRS, or similar)
- Risk levels: Low / Moderate / High (with interpretive text)
- No personal data storage in MVP (results shown but not saved)
