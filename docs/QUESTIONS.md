# ADHD Screening Questions

20 questions adapted from the Adult ADHD Self-Report Scale (ASRS-v1.1) and DSM-5 criteria.

Each question is answered on a frequency scale: Never / Rarely / Sometimes / Often / Very Often.

## Categories

- **inattention** — difficulty sustaining attention, following through, organising tasks
- **hyperactivity** — restlessness, excessive talking, difficulty waiting

---

## Questions
