# Organizational Metrics

Weekly summary of delivery performance and organizational health.

## Key Metrics

### Delivery Gate Compliance
- Target: >95% of outputs meet full delivery gate requirements
- Tracked: Every completed deliverable

### Cycle Time
- Metric: Days from task start → delivery gate pass
- Target: <5 days average
- Tracked: Per deliverable

### Revision Cycles
- Metric: Number of review revisions before shipping
- Target: <1.5 revisions average
- Tracked: Per deliverable

### Documentation Quality
- Target: READMEs are clear, complete, and useful
- Tracked: QA Critic subjective assessment + user feedback

### Handoff Friction
- Metric: Time between agent completion and next agent start
- Target: <4 hours
- Tracked: Per handoff event
